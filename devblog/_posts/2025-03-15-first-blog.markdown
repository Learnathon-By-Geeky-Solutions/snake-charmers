In this post, we will explore the basics of Jekyll and how to use it for creating static websites.

## What is Jekyll?

Jekyll is a static site generator that takes Markdown files and converts them into a static website.

### Setting Up Jekyll

To set up Jekyll, you need to install Ruby and the Jekyll gem. Follow the instructions on the [Jekyll website](https://jekyllrb.com/docs/installation/) to get started.

### Creating a New Post

To create a new post, add a file to the `_posts` directory with the appropriate front matter.

{% highlight ruby %}
---
layout: post
title:  "New Post"
date:   2025-03-16 10:00:00 +0600
categories: blog
---
{% endhighlight %}

### Conclusion

Jekyll is a powerful tool for creating static websites. With a little practice, you can create beautiful and functional websites with ease.
